# Báo Cáo Tiến Độ - Tuần 1

## Dự án
Library Management System (Hệ Thống Quản Lý Thư Viện)

## Thành viên nhóm
- Hoàng Công Vũ
- Trần Phạm Nhất Trung

---

## Công việc đã thực hiện

### Hoàng Công Vũ
- Tạo GitHub repository cho dự án
- Thiết lập cấu trúc project (backend, frontend, database, docs, weekly-reports)
- Viết tài liệu requirements cho hệ thống
- Thiết kế database schema
- Thiết kế tài liệu REST API

Commit links:
- Initial commit
- Project structure setup
- Add requirements document
- Add database design
- Add API documentation

### Trần Phạm Nhất Trung
- Thảo luận và phân tích yêu cầu hệ thống
- Xác định các chức năng chính của hệ thống
- Hỗ trợ thiết kế database và API
